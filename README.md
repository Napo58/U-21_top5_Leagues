# U-21_top5_Leagues
# This project is built aiming at highlighting which nationalities, between the 5 selected, is better at producing young talent.
This project is a screenshot from spring 2023 of the situation of the U-21 football players in the top 5 european leagues, Seriea A, Premier League, Liga, Bundesliga and Ligue 1, to compare the different status of players based on the nationality and the league each played into to better understand in which league the best youngsters play and where they come from
